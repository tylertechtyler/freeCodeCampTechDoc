Getting Started

Hyper text markup language, more commonly referred to as HTML, is used for creating the basic structure of webpages. HTML was made my the World Wide Web Consortium or W3C. The current standard of HTML is known as HTML5. This guide will help you become familiar with some of the basic elements of HTML5.

Page Elements

The most important part of the modern website is it's basic structure. All HTML files will start with the declaration of the document type (shown in all caps). All pages made with HTML will also share the same universal skeleton underneath. These basic elements are listed here:

  • html
  • head
  • body

    • You will also have some structural elements used for dividing up different areas of the page. They are listed as follows:

  • header
  • footer
  • aside
  • article
  • section

    • Below you will see what this html code looks like in its most basic state with the construction of the head and body tags:

    <!DOCTYPE HTML5> <html> <head> </head> <body> <section> <header></header> <article> </article> </section> <footer> </footer> </body> <html>

    Text Elements

    Text elements are the elements that control the way the clearly visible text looks like. These are used to determine the level of importance of text, such as if it is a title, caption, or just a paragraph. You can also specify if you are making bold or italic text. Always remember to use the appropriate tags for your text. You will want to start with making text for your titles and headings. There are six levels of text for headers. Each one will be a different size. They are listed below:

  • h1
  • h2
  • h3
  • h4
  • h5
  • h6

    • To make normal text that one would find in a paragraph, you would just use the <p> tag. You can make an element of text bold by surrounding it with the <strong> tag or italic by surrounding it with the <em> tag. Check out the code snippet below to see how you would do this.

    <html> <body> <h1>This is the Header!</h1> <p>This is the paragraph</p> <p><strong>This text will be BOLD!</strong></p> </body> <html>

    Commenting

    A very important part of html is the ability to write in comments about your code that aren't part of the actual page and won't show up in the browser. A great way to do this is with the built in feature of the html comment. Making comments is very simple.

    <html> <body> <!--Just put the arrow with an exclamation mark followed by two dashes like this--> </body> </html>

    Image and Video

    What would the modern internet be without images and videos? Nothing but text! When you want to add some multimedia excitement to your webpage, all you need to do is link an image or video with the following tags:

    <html> <body> <img src="put-a-link-to-your-image-file-here"> <iframe src="put-a-link-to-your-video-file-here"></iframe> </body> </html>

    The great thing about the <img> tag is that it is self closing. The <iframe tag> however, still needs to be closed. You can also set the size of these elements by putting "width=" and "height=" and then the dimensions you want in pixels in quotation marks.


    Making Links

    Being able to link to other webpages is a very useful part of html. To make a link you use the >a> tag. The most important part to remember is the href attribute, which points to the source of the link. Links will appear blue and underline by default, and then they will turn purple when they have been visited.

    <html> <body> <a href="this-is-where-you-put-the-url-of-the-site-that-your-link-goes-to"</a> </body> </html>

    Making Lists

    You may find yourself wanting to make lists on your webpage at some point. Html has the ability to make these lists, either with bullet points or numbers. In order to make a list you would use the <li> tag. To specify whether it is bulleted or numbered you would use the <ul> meaning "unordered list" or the <ol> tag meaning "ordered list". Lists also come in handy for making menu and navigation bars.

    <html> <body> <ol> <li>Item one</li> <li>Item two</li> <li>Item three</li> </ol> <ul> <li>Something</li> <li>Anything</li> <li>This thing</li> </body> </html>

    Forms and Fields

    The ability to add forms is one of html's most useful features. A form will display areas on a webpage where a user can input information. This is very important, especially if you want your site to collect information from a visitor such as a name or email address. There are many possible form elements that one can put into the <form> tag. Some of them are listed below:

  • input
  • textarea
  • button
  • select
  • option
  • label
  • output

    • Have a look at the code snippet below to see how you would go about creating a basic form.

    <html> <body> <form> <label>Name</label> <input type="text"> <label>Email</label> <input type="email"> <button type="submit">Submit!</button> </form> </body> </html>
